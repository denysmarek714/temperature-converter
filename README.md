# Temperature Converter v1.1

## Короткий опис
**Temperature Converter** — це зручний консольний інструмент на Python для швидкої конвертації температури. Програма забезпечує точні перетворення між градусами Цельсія, Фаренгейта та Кельвінами, має інтуїтивно зрозумілий інтерфейс та вбудований захист від помилок введення.

## Функціональність
Програма надає користувачеві меню з такими можливостями:
* **Цельсій → Фаренгейт** (конвертація `°C` у `°F`)
* **Фаренгейт → Цельсій** (конвертація `°F` у `°C`)
* **Цельсій → Кельвін** (конвертація `°C` у `K`)
* **Кельвін → Цельсій** (конвертація `K` у `°C`)

**Особливості роботи:**
* Результати автоматично округлюються до двох знаків після коми для зручності.
* Реалізовано перехоплення помилок (наприклад, якщо введено текст замість числа).

## Інструкції з встановлення
Дотримуйтесь цих кроків, щоб запустити проєкт:
1.  Переконайтеся, що на вашому комп'ютері встановлено Python (версія 3.6 або вище).
2.  Завантажте файл `converter.py` у папку вашого проєкту.
3.  Відкрийте термінал (або командний рядок) у цій папці.
4.  Запустіть скрипт, використовуючи наступну команду:
python converter.py

## Приклад використання
=== Конвертер температур ===
1. Цельсій -> Фаренгейт
2. Фаренгейт -> Цельсій
3. Цельсій -> Кельвін
4. Кельвін -> Цельсій

Виберіть операцію (1-4): 1
Введіть температуру: 36.6
